Redow.Core

Мінімалістична CSS-бібліотека для створення інтерфейсів з ефектом засклення. Легка, швидка та адаптивна.

Архітектура

Система базується на нативних властивостях CSS3. Вона не потребує JavaScript для рендерингу основних ефектів, що забезпечує максимальну швидкість роботи інтерфейсу.

Впровадження

<link rel="stylesheet" href="redow.css">
