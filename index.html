Redow.

Спеціалізована CSS-бібліотека для створення інтерфейсів з використанням сучасних графічних фільтрів та ефекту матового скла.

Встановлення

Для використання бібліотеки додайте наступний рядок у блок <head> вашого проекту:

<!-- Підключення через CDN -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ВАШ_АККАУНТ/redow/redow.css">

* Ви також можете завантажити файл безпосередньо для локального хостингу.
